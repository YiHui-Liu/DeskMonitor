# ESP32主控参数
* 模组：ESP32-32E
* CPU ：ESP32-D0WD-V3,Xtensa双核32位LX6 微处理器
* 主频：240MHz（最大）
* 存储：448KB ROM+520KB SRAM+16KB RTC SRAM+4MB外接QSPI Flash
* WIFI ：2.4GHz、802.11b/g/n模式
* 蓝牙：蓝牙 V4.2 BR/EDR 和蓝牙LE标准
* 工作电压：3.0~3.6(V)
* 工作温度：–40~85(℃)

# 液晶屏参数
* 屏幕尺寸：4.0 inch
* 屏幕类型：TN TFT
* 分辨率：320xRGBx480(pixels)
* 有效显示区：55.68(W)x83.52(H)(mm)
* 颜色数目：最大：262K(RGB666)
* 常用：65K(RGB565)
* 驱动IC：ST7796S
* 显示接口：4-Line SPI(接到ESP32上)
* 像素尺寸：0.174(H)x0.174(mm)
* 可视角度：12 0’CLOCK
* 背光亮度(典型值)： 450 cd/m2
* 背光灯类型：White LED*8
* 工作温度：-20~60(℃)
* 存储温度：-30~70(℃)

# 尺寸参数
* 液晶屏外形尺寸：60.88±0.2(W)x94.57±0.2(H)x2.5±0.1(D)(mm)(不含排线和背胶)
* 模块外形尺寸：60.88(W)x111.11(H)x4.60(D)(mm)

# 电气参数
* 工作电压：5.0V
* 背光电流：104mA
* 总电流SP32复位时：40mA
* 仅有显示屏工作时：230mA
* 显示屏、喇叭、电池充电都工作时：580mA
* 功耗：1.15W（只有显示屏工作）
* 亮度：470 cd/m2
* 锂电接口充电电压范围：4.2~6.5(V)
* 典型值：5V
* 充电电流：最大值：500mA
* 模块实际值：290mA
* 建议外接电池规格：3.7V聚合锂电池
* 喇叭接口参数(最大)：1.5W(8Ω)或 2W(4Ω)
